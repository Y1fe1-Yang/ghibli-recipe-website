# 🎨 漫画食谱生成器使用指南

## ✨ 核心功能

这是一个**零登录**、**AI驱动**的漫画风格食谱生成器。每道菜的**每个烹饪步骤**都会生成一张吉卜力风格的漫画插图！

### 🎯 工作流程

```
搜索菜名 → AI生成食谱内容 → 生成主菜图片 → 为每个步骤生成漫画图片 → 完成！
```

## 🚀 快速开始

### 访问地址
**https://3000-capy-1769767690860-182316-preview.happycapy.ai**

### 使用方法

1. **打开网站** - 无需注册或登录
2. **搜索菜名** - 在大搜索框输入任何菜名
3. **等待生成** - AI会自动：
   - 生成详细的食谱内容（15秒）
   - 生成完成品的吉卜力风格插图（30秒）
   - **为每个烹饪步骤生成漫画图片**（每步30秒）
4. **查看漫画食谱** - 点击卡片查看带插图的完整步骤

## 💡 使用技巧

### 1. 菜名输入技巧
- ✅ **具体菜名**："番茄炒蛋"、"红烧肉"
- ✅ **带菜系**："川味麻婆豆腐"、"广式早茶"
- ✅ **创意组合**："抹茶慕斯"、"芝士焗饭"
- ❌ 避免太笼统："炒菜"、"汤"

### 2. 生成时间
- **单个食谱总时间**：约 2-5分钟
  - 食谱内容：15秒
  - 主菜图片：30秒
  - 每个步骤图片：30秒 × 步骤数量

**注意**：8个步骤的菜谱大约需要5分钟完成全部图片生成

### 3. 最佳体验
- **简单菜肴优先**：初次使用建议选择5步以内的简单菜肴
- **等待完成**：生成过程中会显示进度，请耐心等待
- **刷新查看**：生成完成后页面会自动刷新并显示新食谱
